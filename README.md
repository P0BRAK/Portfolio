This repository contains the source code for my professional portfolio website, showcasing my skills and projects in data analysis.

## 🚀 Projects

### 1. Churn Modeling in Telco
Predictive analysis to identify customers at risk of churning using behavioral data and usage patterns.
- **Tech Stack:** Python, XGBoost
- **Performance:** 89% Accuracy

### 2. Stock Price Regression
Regression models to analyze historical stock trends and predict future price movements with error margins.
- **Tech Stack:** Python, Scikit-Learn (Random Forest)
- **Performance:** 1.2% MAE

### 3. Customer Segmentation
Advanced clustering techniques to group customers into meaningful segments for targeted marketing strategies.
- **Tech Stack:** K-Means, PCA, Python
- **Outcome:** Identified 5 distinct customer personas
