# Project Plan: Data Analyst Portfolio Website

## Objective
Build a professional, modern data analyst portfolio on GitHub Pages showcasing personal branding, three key projects, and contact information.

## Design Specifications
- **Theme:** Modern Dark Theme.
- **Typography:** JetBrains Mono (monospace aesthetic).
- **Structure:**
  - **Hero:** Name ("Pobrak"), professional title.
  - **Projects:** Section showcasing three data analysis projects.
  - **Contact:** Section with professional contact details (GitHub, LinkedIn, Email).

## Implementation Steps

### 1. Setup & Project Structure
- Initialize a local folder for the website.
- Create base file structure:
  - `index.html`: Main layout and content.
  - `style.css`: Dark theme and JetBrains Mono integration.
  - `script.js`: (Optional) Interactivity for project previews.
  - `README.md`: Project documentation.

### 2. Development Phase
- **HTML:** Semantic structure for Hero, Project Cards, and Contact section.
- **CSS:**
  - Apply global dark theme colors.
  - Import and configure JetBrains Mono via Google Fonts.
  - Implement responsive Flexbox/Grid layout for project cards.
- **Content:** Populate project details and placeholders for links/images.

### 3. Version Control & Deployment
- Initialize Git repository.
- Push codebase to GitHub.
- Configure GitHub Pages settings to deploy from the `main` branch.

### 4. Verification
- Validate the site live on `https://[username].github.io/[repo-name]/`.
- Ensure mobile responsiveness and correct font rendering.
